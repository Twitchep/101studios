# 🖼️ Image URLs - Quick Reference Card

## Copy-Paste These Instructions:

### 1️⃣ **Get an Image URL (3 Free Options)**

#### **ImgBB (Easiest - Recommended!)**
- Go to: https://imgbb.com/
- Click "Start uploading"
- Select your image
- Copy the "Direct Link" URL
- Paste in editor

#### **Imgur**
- Go to: https://imgur.com/
- Click "New Post" → "Images/Videos"
- Select image
- Right-click → "Copy image address"
- Paste in editor

#### **Local Files**
- Put image in: `public/images/`
- Use in editor: `/images/your-image.jpg`

---

## 2️⃣ **Add Image in Visual Editor**

1. Open: http://localhost:8082/editor.html
2. Find the item
3. Paste URL in "Image URL" field
4. See preview below
5. Click "💾 Save All Changes"
6. Done! ✅

---

## 3️⃣ **Image URL Format Examples**

```
https://i.ibb.co/abc123/portfolio-design.jpg
https://imgur.com/xyz789
/images/product.jpg
/images/portfolio-item.png
```

---

## 🎯 **Image Tips**

✅ **Good URLs:**
- `https://i.ibb.co/xxxxx/image.jpg`
- `https://imgur.com/xxxxx`
- `/images/my-image.jpg`

❌ **Bad URLs:**
- `http://...` (use https://)
- `C:\Users\...` (local computer paths won't work)
- Missing file extension
